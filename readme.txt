Steps and Dependencies

1. Create a folder - name is the project name.
2. Open visual code. Open the project folder.
3. npx create-react-app . 
4. Google font. Nunito. 
5. npm i sass
6. npm i react-router-dom (https://reactrouter.com/)
7. Material Icons  
    npm install @mui/material @emotion/react @emotion/styled
    npm install @mui/icons-material    
    import DashboardIcon from '@mui/icons-material/Dashboard';
8. box-shadow (https://html-css-js.com/css/generator/box-shadow/)
9. npm install --save react-circular-progressbar
10. npm install recharts (https://recharts.org/en-US/api)
11. npm install @mui/x-data-grid (https://mui.com/x/react-data-grid/)
